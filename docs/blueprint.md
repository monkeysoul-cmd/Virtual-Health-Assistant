# **App Name**: HealthWise Chat

## Core Features:

- Interactive Symptom Checker: An AI-powered symptom checker that identifies probable diseases based on user-inputted symptoms, acting as a tool.
- Precautionary Advice: Provides general precautionary measures and advice for the health conditions identified by the symptom checker.
- Health Test Suggestions: Suggests relevant health tests that are useful for diagnosing the probable diseases identified.
- Local Doctor Directory: Provides a searchable demo list of doctors with contact details, filtered by area.

## Style Guidelines:

- Primary color: Soft green (#8FBC8F) to promote calmness and health.
- Background color: Off-white (#F5F5DC), almost the same hue, for a clean and soothing backdrop.
- Accent color: A slightly more saturated green (#90EE90), analogous to the primary, for interactive elements and highlights.
- Body and headline font: 'PT Sans' for readability and a modern, friendly feel.
- Use simple, easily recognizable icons for symptoms, tests, and medical specialties.
- Clean, well-spaced layout with a focus on readability and ease of navigation.
- Subtle animations for loading states and transitions to provide a smooth user experience.
