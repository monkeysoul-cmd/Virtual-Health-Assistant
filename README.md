# Health Assistant

A React + Next.js health assistant app converted to JavaScript.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Start production server:

```bash
npm start
```

## Notes

- Path aliases are configured via `jsconfig.json`.
- The app uses `cross-env` for cross-platform environment variable support.
